1. 什么是高阶函数和柯理化并举例说明。
函数的形参是一个函数或者返回值是一个函数，则此函数为高阶函数。
柯里化是指将一个多参函数转化为一个多层调用的函数的过程。比如:fn(x,y)转化为fn(x)(y) 的形式。
def fn(x,y)
	ret = x + y
	return ret
转发方式为：
def fn(x)
	def _fn(y)
		ret = x + y 
		return ret
	return _fn
增加一个函数成为嵌套函数，将原有函数体右移，成为新函数的函数体，原函数返回此新函数，将原函数的后面参数下移成为新函数的参数。


 2. 请列出functools包内的每个函数的功能作用。
 update_wrapper： 默认接收两个位置参数，一个为装饰函数，一个被装饰函数，将装饰函数的相关属性信息替换为被装饰函数的属性信息。
 wraps：默认接收一个位置参数，为被装饰函数，将装饰函数的相关属性信息替换为被装饰函数的属性信息。
 partial：偏函数，使其可以将函数的两个或多个参数固定一部分，使其变成接收单个参数的一个函数。
 lru_cache：如果同样的参数得到的结果肯定一致，则可以使用其将结果缓存，未来再次调用，可以直接利用其结果。
 
 4. 什么是类型注解并说明其背后的目的
 因为python是强类型语言，赋值即定义，赋值及重新定义，一个函数只希望接收某个类型的参数，或者只希望返回某个类型的返回值。
 可以使用类型注解，仅能进行提示，并不能强制。但是可以编写装饰器起到提示和抛异常的功能。
 使用inspect模块，获得函数签名，其paramers的方法可以获得函数的相关属性信息，其中包括函数定义的类型注解。
